Partial Execution

                
Select a portion of SQL in the editor and press Ctrl+Enter
                   (or click Execute) to run only the selected text.
                   This is useful for executing one statement at a time from a multi-statement script.

ExecuteRun SQL as-is

                        
EXPLAINShow query plan (no execution)

                        
ANALYZEEXPLAIN ANALYZE (execute + actual timings)
