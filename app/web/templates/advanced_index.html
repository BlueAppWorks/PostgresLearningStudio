PostGIS
{{ extensions.get('postgis', {}).get('installed') or extensions.get('postgis', {}).get('available', 'N/A') }}

PostgreSQL に地理空間データの機能を追加するエクステンション。 地図上の座標（ポイント・ライン・ポリゴン）を DB に格納し、距離計算や包含判定などの空間クエリを SQL で実行できます。

  • ポイント・ライン・ポリゴンの格納と描画
  • 距離計算・面積計算・包含判定
  • GiST 空間インデックスによる高速検索

開発: PostGIS Project (OSGeo) / ライセンス: GPL v2

pg_hint_plan
{{ extensions.get('pg_hint_plan', {}).get('installed') or extensions.get('pg_hint_plan', {}).get('available', 'N/A') }}

SQL コメントにヒントを書くことで、PostgreSQL のクエリプランナーに実行計画を指示できるエクステンション。 統計情報が古い場合や、プランナーが最適でない計画を選ぶ場合に有効です。

  • SeqScan / IndexScan の強制指定
  • JOIN 方式の指定（NestLoop / HashJoin / MergeJoin）
  • 古い統計情報による非効率な計画の修正

開発: NTT OSS Center DBMS Development and Support Team / ライセンス: PostgreSQL License

GitHub (ossc-db/pg_hint_plan) Demo
pgvector
{{ extensions.get('vector', {}).get('installed') or extensions.get('vector', {}).get('available', 'N/A') }}

PostgreSQL にベクトル型とベクトル検索を追加するエクステンション。 テキストを数値ベクトルに変換し、意味的に似た文書を検索する「セマンティック検索」を SQL で実現できます。AI/RAG の基盤技術です。

  • vector 型によるベクトル格納
  • コサイン類似度・L2距離による検索
  • HNSW / IVFFlat インデックスで高速近傍検索

開発: Andrew Kane / ライセンス: PostgreSQL License

PostgreSQL に Apache Iceberg とデータレイクアクセス機能を追加する Snowflake 製エクステンション。 CREATE TABLE ... USING iceberg で Iceberg テーブルを作成し、S3 上の Parquet/CSV を直接クエリできます。
