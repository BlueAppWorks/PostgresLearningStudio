pgvector Demo

テキストをベクトルに変換し、意味的に似た文書を検索する「セマンティック検索」を体験します。 pgvector は PostgreSQL にベクトル型とベクトル検索を追加する OSS エクステンションです。

仕組み

    

        
このデモでは、ビジネス関連のキーワード辞書を使った簡易ベクトル化を行います。

           （本番環境では OpenAI Embeddings や Snowflake Cortex などの AI モデルでより高精度なベクトルを生成します）

        
キーワード辞書（20次元ベクトル）:

        

            商談契約
            提案見積
            受注失注
            訪問電話
            メール会議
            新規既存
            競合価格
            納期品質
            サポート導入
            検討決裁
        

        
各キーワードの出現回数を数えてベクトル化し、L2正規化します。

    







    
Step 1: 既存の営業日報を確認

    

        
-- 営業日報の一覧（ベクトルは省略表示）
SELECT id, diary_date, salesperson, company,
       LEFT(content, 50) || '...' AS content_preview,
       embedding IS NOT NULL AS has_vector
FROM vector_demo.sales_diary ORDER BY id;

Step 3: ベクトル化 (UPDATE)

    

        
INSERT した行のテキストをベクトルに変換します。embedding IS NULL の行のみが対象です。

        
-- 未ベクトル化の行をベクトル化
UPDATE vector_demo.sales_diary
SET embedding = vector_demo.text_to_vector(content)
WHERE embedding IS NULL;

-- ベクトル化結果を確認（最新の行）
SELECT id, LEFT(content, 40) AS content_preview,
       embedding::text AS vector
FROM vector_demo.sales_diary
ORDER BY id DESC LIMIT 1;
